# Security Policy

## Reporting a Vulnerability

We take the security of Craken seriously. If you discover a security vulnerability in this project, please do not disclose it publicly. Instead, please report it to the repository owner at [aungminkhant.shay@gmail.com](mailto:aungminkhant.shay@gmail.com).

## Supported Versions

Only the latest stable version of Craken is supported with security updates.

## Response Time

We aim to respond to all security reports within 48 hours and provide a fix as soon as possible.
